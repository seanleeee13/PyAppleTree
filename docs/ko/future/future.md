# 추후 계획

## 수정 예정 버그

- 실시간 로그 메세지가 터미널에서 밀려서 도배가 되는 버그
- 분석 불가능한 경우가 정상 함수처럼 보이는 버그
- 점유율이 낮은 재귀함수가 실행의 입구라고 표시되는 버그

## 추가 예정 기능

- 병목 여부를 상대평가로 변경
- 분석 결과 확신도 표시
- 용어 수준 조정
- 샘플 수 경고 기준이 동적으로 변하도록 조정
- 영어 지원 확장
- 리포트 표로 보기 기능 추가
- 기본 `profiling.sampling` 기능 추가
- 리포트를 파일로 저장 기능 추가
- 특정 함수만 분석하기 기능 추가
- 함수 다중 유형 표시 기능 추가
