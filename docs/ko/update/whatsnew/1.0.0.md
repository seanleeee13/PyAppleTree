# 1.0.0 신규 기능

## 주요 사항

### 줄 분석에서 보여주는 줄의 수를 동적으로 변경하도록 수정

줄 분석에서 보여주는 줄의 수를 3개 이상 10개 이하로 동적으로 변경되도록 수정했습니다. 내부 로직에서는 줄의 수가 3~10개가 되는 한도 내에서 줄을 점유율 순으로 정렬했을 때 점유율 격차가 가장 커지는 지점까지 줄을 보여주도록 설정했습니다.

### 각종 로직 최적화

통계 계산 로직과 리포트 생성 로직에서 동일 코드 다회 반복이나 비효율적 로직을 전면 수정했습니다. 그 결과, 오버헤드 7%로 `py-spy`와 비슷한 성능을 보였습니다.

### `advanced`, `lab` 모드 추가

상세 분석 모드인 `advanced` 모드와 실험 기능이 포함된 `lab` 모드를 만들고 기존 기능을 `advanced`와 `lab`으로 분류했습니다.

## 기타 신규 기능

- Python `3.15.0b2` 버전까지 지원을 확장했습니다.
- 실시간 로그가 겹치는 등의 현상을 수정했습니다.
